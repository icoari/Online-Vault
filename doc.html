Documentation du Gestionnaire de mots de passe

1. Présentation Générale

Ce gestionnaire de mots de passe vous permet de créer, stocker, catégoriser et chiffrer vos identifiants de façon sécurisée. Il fonctionne côté client, ce qui signifie que vos données ne quittent pas votre machine.

2. Fonctionnalités Clés

  • Chiffrement AES-GCM : tous les mots de passe sont protégés.
  • Organisation par catégorie : classez facilement vos identifiants.
  • Recherche et filtrage : retrouvez rapidement vos comptes.
  • Export/Import : sauvegardez et restaurez votre Vault au format HTML chiffré.
  • Analyse de sécurité : identifiez les mots de passe faibles ou compromis (via l'API HaveIBeenPwned).
  • Gestion 2FA (OTP) : ajoutez la génération de codes à 6 chiffres pour une authentification à deux facteurs.

3. Utilisation

3.1 Créer une nouvelle session

Sur la page d'accueil, cliquez sur "Créer une nouvelle session", puis choisissez un mot de passe maître suffisamment robuste (au moins 12 caractères, comprenant majuscules, chiffres et symboles).

3.2 Importer un fichier existant

Si vous disposez déjà d'un fichier HTML chiffré (exporté préalablement), cliquez sur "Importer un fichier chiffré", glissez votre fichier ou utilisez le bouton "Choisir un fichier" puis saisissez le même mot de passe maître qui avait été utilisé pour créer ce fichier.

3.3 Ajouter des entrées

Une fois le Vault déverrouillé, vous pouvez ajouter de nouvelles entrées (service, identifiant, mot de passe, catégorie, etc.) via le bouton "Nouveau". Il est possible de générer automatiquement un mot de passe aléatoire ou de copier/coller un existant.

3.4 Exporter et sauvegarder

Pour éviter toute perte, vous pouvez régulièrement exporter votre Vault via le bouton "Exporter". Cela génère un fichier HTML chiffré (uniquement lisible avec le mot de passe maître adéquat).

4. Raccourcis Clavier

  • Alt + N : Ouvrir la fenêtre pour ajouter un nouveau mot de passe.
  • Alt + E : Lancer l’export du Vault.
  • Alt + L : Verrouiller immédiatement le Vault.
  • Échap (Esc) : Fermer la plupart des fenêtres/modales ouvertes.

5. Analyse de sécurité

Le gestionnaire propose un outil d'analyse qui détecte :

  • Les mots de passe trop courts ou faibles (manquant de variété).
  • Les mots de passe déjà compromis dans des fuites de données publiques (via HaveIBeenPwned).
  • Les mots de passe réutilisés sur plusieurs services.

Il vous est alors conseillé d’améliorer ou de changer ces mots de passe afin de mieux protéger vos comptes.

6. Authentification à Deux Facteurs (OTP)

Vous pouvez configurer une clé secrète OTP (format Base32) pour générer des codes à 6 chiffres, renouvelés toutes les 30 secondes. Cela permet de renforcer la sécurité d'un compte.

7. Questions Fréquentes (FAQ)

7.1 Puis-je utiliser n'importe quel mot de passe maître ?

Techniquement oui, mais il est fortement recommandé d’en choisir un suffisamment long et complexe pour résister à d’éventuelles attaques. Évitez d'utiliser un mot de passe déjà utilisé ailleurs.

7.2 Que faire en cas d'oubli du mot de passe maître ?

Malheureusement, si vous oubliez votre mot de passe maître, il sera impossible de récupérer le contenu de votre Vault, car tout est chiffré de manière sécurisée. Il est donc primordial de le retenir ou de le stocker en lieu sûr.

7.3 Mes données sont-elles envoyées sur un serveur externe ?

Non. Toutes les opérations de chiffrement/déchiffrement se font localement, dans votre navigateur. Les données ne sortent pas de votre poste, sauf si vous choisissez de les exporter vous-même (fichier HTML).
